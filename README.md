# 智能计算课题组网站

一个现代化的课题组网站，具有简洁高级的设计风格，支持在线编辑和管理功能。

## 🚀 功能特性

### 网站页面
- **首页**: 课题组概览、统计数据、快速导航
- **关于我们**: 课题组介绍、使命愿景
- **团队成员**: 人员展示、分类筛选、点击查看详细介绍
- **个人介绍页**: 详细的个人履历、教育背景、工作经历、学术成果
- **研究方向**: 研究领域介绍
- **学术成果**: 论文展示、类型筛选
- **新闻动态**: 最新研究进展
- **联系我们**: 联系信息和反馈表单

### 管理功能
- **安全访问**: 密码保护的管理后台，隐藏入口，只有授权人员可访问
- **成员管理**: 添加、编辑、删除团队成员（包含详细信息：教育背景、工作经历等）
- **论文管理**: 添加、编辑、删除学术成果
- **系统设置**: 网站基本信息配置
- **本地存储**: 数据持久化保存
- **个人页面**: 自动生成详细的个人介绍页面

### 设计特色
- **响应式设计**: 完美适配各种设备
- **创新活力**: 现代化UI设计，渐变色彩
- **交互友好**: 流畅的动画和过渡效果
- **易于维护**: 模块化代码结构

## 🛠️ 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **样式**: 自定义CSS，Font Awesome图标
- **字体**: Google Fonts (Inter)
- **存储**: LocalStorage
- **兼容性**: 现代浏览器

## 🚀 快速开始

### 本地运行

1. 克隆或下载项目文件
2. 在浏览器中打开 `index.html`
3. 网站即可正常运行

### 部署到 GitHub Pages

1. 创建 GitHub 仓库
2. 将项目文件推送到仓库
3. 在仓库设置中启用 GitHub Pages
4. 选择主分支作为发布源
5. 访问生成的 GitHub Pages 链接

### 配置自定义域名（可选）

1. 在项目根目录创建 `CNAME` 文件
2. 写入你的域名，如：`research.youruniversity.edu.cn`
3. 在域名提供商处配置 DNS 解析
4. 在 GitHub 仓库设置中绑定自定义域名

## ⚙️ 配置说明

### 初始数据
网站首次加载时会自动创建示例数据，包括：
- 6个团队成员（教授、博士生、硕士生、博士后）
- 5篇学术论文（期刊论文、会议论文、预印本）

### 自定义配置
通过管理后台可以修改：
- 课题组名称和描述
- 联系信息
- 团队成员信息
- 学术成果信息

### 数据管理
所有数据存储在浏览器本地存储中，支持：
- 数据的增删改查
- 页面刷新后数据保持
- 清除浏览器数据会重置为初始状态

## 🎨 自定义样式

### 修改配色方案
在 `css/styles.css` 中修改 CSS 变量：

```css
/* 主要颜色 */
--primary-color: #2563eb;
--secondary-color: #7c3aed;
--accent-color: #ffd700;
```

### 修改字体
在 `index.html` 的 `<head>` 部分修改 Google Fonts 引用：

```html
<link href="https://fonts.googleapis.com/css2?family=Your-Font&display=swap" rel="stylesheet">
```

## 📱 响应式设计

网站支持以下屏幕尺寸：
- **桌面端**: 1200px+
- **平板端**: 768px - 1199px
- **手机端**: 480px - 767px
- **小屏幕**: < 480px
